# OrchestraBench — DAI 2026 submission (LaTeX)

Full-content LaTeX draft converted 1:1 from the repo's `PAPER.md` (v0.6). This is
the **"write the full draft first, then cut to the page limit"** stage Spurthi
asked for — it is **not yet page-trimmed and not yet anonymized**.

## Files
- `main.tex` — ACM `acmart` (`sigconf`, `nonacm` draft mode); abstract + §1–§10 +
  6 tables + 2 figures.
- `references.bib` — 13 BibTeX entries (every arXiv ID and missing author list is
  flagged `VERIFY` — check before submission; do not submit unverified refs).
- Figures are pulled from the repo's top-level `figures/` via
  `\graphicspath{{../figures/}{./figures/}{./}}`.

## Compile (Overleaf — this machine has no LaTeX toolchain)
1. New Project → Upload Project, or create blank and add `main.tex` +
   `references.bib`.
2. Upload `figures/exp3_cascade_by_depth.png` and `figures/exp2_arith_vs_domain.png`
   (either next to `main.tex` or in a `figures/` subfolder — `\graphicspath`
   covers both).
3. `acmart` is built into Overleaf; set the compiler to **pdfLaTeX**. Compile runs
   `pdflatex → bibtex → pdflatex ×2`.
